<util/parity.h>: Parity bit generation

Detailed Description

 #include <util/parity.h>

This header file contains optimized assembler code to calculate the parity bit for a byte.

(__extension__({                                        \

        __asm__ (                                       \
                "mov __tmp_reg__,%0" "\n\t"             \
                "swap %0" "\n\t"                        \

                "mov __tmp_reg__,%0" "\n\t"             \
